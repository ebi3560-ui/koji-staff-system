# 工事部スタッフ一元管理システム

パチンコ・スロット台設置業務に特化したスタッフ管理Webアプリです。

## 主な機能

| 機能 | 説明 |
|------|------|
| スタッフ一覧 | 登録済みスタッフの検索・閲覧・編集 |
| 新規登録 | スタッフ情報・シフト希望・書類の登録 |
| シフト確認 | 月次・週次カレンダーでシフトを管理 |
| リマインド | 書類期限・更新通知の管理 |
| 現場管理 | 現場ごとのスタッフ配置・ドラッグ&ドロップ割当 |
| 給与管理 | 勤務実績・給与明細の管理 |
| 請求書 | 取引先向け請求書の作成・印刷 |

## 使い方

ブラウザで `index.html` を開くだけで動作します。サーバー不要です。

## 動作環境

- モダンブラウザ（Chrome / Edge / Safari / Firefox 最新版）
- スマートフォン対応（レスポンシブデザイン）
